# Changelog

## 0.2.0 - 2026-03-08

Operational release of Memento Kit as an agent work operating system.

Includes:

- one-line installer via `install.sh`
- project-truth bootstrap flow via `npm run adopt:bootstrap`
- generated `project-truth-bootstrap.md`
- git workflow layer and repo-local git config bootstrap
- English `MANUAL.md`
- English and Korean post-install checklists
- English-first README ordering and GitHub-facing packaging
- validated install flow that reaches `docs:check` on a fresh repo

## 0.1.0 - 2026-03-07

Initial public packaging of Memento Kit.

Includes:

- repo bootstrap via `setup.sh`
- canonical context routing docs
- generated route/store/API maps
- semantic checkpoint, brief, handoff flows
- context validation and governance checks
- local harness scripts
- context value demo report
- agent catalog and tool catalog
- local registry and retrieval APIs
- telemetry, A/B reporting, and coordination helpers
- Korean quick manual for first-time adopters
